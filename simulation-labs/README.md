# Simulation Labs

Labs de simulacao guiada para transformar leitura em decisao com consequencia visivel.

## Como usar

1. leia o cenario;
2. mexa nos controles no cockpit;
3. explique o que quebra primeiro;
4. responda o mastery check;
5. salve a tentativa se a decisao foi ruim ou hesitante.
