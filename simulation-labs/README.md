# Simulation Labs

Labs de simulacao guiada para transformar leitura em decisao com consequencia visivel.

## Como usar

1. escolha o lab sugerido no `Study Context` do chapter;
2. leia o cenario;
3. mexa nos controles no cockpit;
4. explique o que quebra primeiro;
5. responda o mastery check;
6. depois puxe o playbook ou bridge lab do mesmo chapter para transformar intuicao em rotina.
