# Decision Contrasts

Treino de discriminacao entre opcoes que parecem primas, mas resolvem dores diferentes.

## Como usar

1. leia so o titulo do contraste
2. tente responder em voz alta:
   - qual problema puxa a opcao A?
   - qual problema puxa a opcao B?
   - qual erro comum faz a equipe confundir as duas?
3. so depois abra o gabarito
