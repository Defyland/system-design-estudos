# Reviews - Backend Interview Foundations

Revisao recall-first para a trilha paralela de entrevistas backend.

## Como usar

1. responda tudo em voz alta
2. puxe o `Cue Signal` antes do resto
3. tente falar tempo, espaco e trade-off sem olhar
4. marque mentalmente cada card como:
   - `liso`
   - `hesitante`
   - `errei`
