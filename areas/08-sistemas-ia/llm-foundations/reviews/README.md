# Reviews - LLM Foundations

Revisao recall-first para a trilha paralela de fundamentos de LLM.

## Como usar

1. responda tudo em voz alta
2. puxe o `Cue Signal` antes do resto
3. olhe o gabarito logo em seguida
4. marque mentalmente cada card como:
   - `liso`
   - `hesitante`
   - `errei`
