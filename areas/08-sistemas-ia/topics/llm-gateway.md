# LLM Gateway

## When to Use

Use quando varios modelos, tenants, budgets, policies e fallbacks precisam de uma fronteira unica.

## What Breaks First

Custo por token e latencia ficam invisiveis se cada feature chama modelo direto.

## Interview Trap

Gateway nao substitui avaliacao de qualidade.
