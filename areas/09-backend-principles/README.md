# 09 - Backend Principles

## Por que esta area existe

Backend Principles cobre fundamentos de backend que aparecem antes de system design: HTTP, routing, contratos de API, auth, validacao, persistencia, cache, jobs, observabilidade, shutdown, seguranca e performance.

## Como estudar

Use esta area para fechar gaps praticos. Para cada card:

1. explique o conceito em voz alta;
2. desenhe o caminho do request;
3. diga o que quebra primeiro;
4. implemente ou revise um exemplo pequeno em Rails, Elixir ou Go.
