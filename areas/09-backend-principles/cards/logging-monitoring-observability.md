# Logging, Monitoring and Observability

## When to Use

Use para responder rapidamente: o que aconteceu, onde, para quem, com qual impacto e qual primeira mitigacao.

## What Breaks First

Log demais sem estrutura, metrica sem baseline e trace sem correlation id deixam incidente dependente de sorte.

## Interview Trap

Falar "tem dashboard" sem primeira metrica. Observabilidade boa aponta sintoma, escopo, causa provavel e rollback.

## Practice Drill

Para `POST /payments`, defina tres logs estruturados, quatro metricas, um trace id e o alerta que acorda alguem.
