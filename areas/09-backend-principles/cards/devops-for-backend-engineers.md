# DevOps for Backend Engineers

## When to Use

Use para conectar codigo a operacao: build, deploy, rollback, migration, infra, secrets, observabilidade, incidentes e capacidade.

## What Breaks First

Backend funciona localmente, mas deploy quebra por config, migration irreversivel, falta de health check ou rollback manual demais.

## Interview Trap

Dizer "isso e com infra". Backend senior sabe o suficiente para tornar release segura e diagnosticar falha no caminho de producao.

## Practice Drill

Desenhe release de endpoint novo: feature flag, migration backward-compatible, health check, metricas, canary e rollback em menos de 5 minutos.
