# Network Partitions

## When to Use

Use particoes para pensar quando dois lados do sistema continuam vivos, mas nao conseguem conversar com confianca.

## Interview Trap

Tratar particao como downtime total. Muitas falhas perigosas acontecem quando os dois lados continuam aceitando trabalho.
