# Consensus

## When to Use

Use consenso quando replicas precisam concordar sobre estado ou lideranca mesmo com falhas.

## Interview Trap

Consenso e caro. Se o problema e deduplicar efeito local, uma chave idempotente pode resolver com menos operacao.
