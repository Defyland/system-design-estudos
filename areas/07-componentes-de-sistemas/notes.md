# Notes

## Regra de decisao

Um componente entra quando reduz uma dor concreta: latencia, blast radius, repeticao de politica, isolamento, observabilidade ou custo. Se ele so deixa o desenho mais bonito, ainda nao entrou.

## Perguntas antes de adicionar componente

- que requisito fica menos burro se eu remover essa peca?
- qual falha ela isola?
- qual nova falha ela cria?
- quem opera, monitora e reverte?
