# API Gateway

## When to Use

Use quando autenticacao, roteamento, rate limiting ou transformacao leve precisam ficar numa fronteira comum.

## What Breaks First

Politica demais no gateway transforma borda em monolito invisivel.

## Interview Trap

Gateway nao substitui autorizacao de dominio.
