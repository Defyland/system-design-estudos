# GitHub - Rails and MySQL at Scale

## Why this case matters

Mostra como um monolito Rails antigo continua competitivo quando a engenharia de banco, upgrades e operacao e madura.

## Course topics

- bancos SQL
- manutencao e upgrades
- leitura, escrita e particionamento
- monolitos
- deploys e escalabilidade

## Stack relevance

- Rails: very high
- Elixir: low
- Go: medium

- [Upgrading GitHub.com to MySQL 8.0](https://github.blog/engineering/infrastructure/upgrading-github-com-to-mysql-8-0/)

- por que o monolito ainda funciona
- disciplina de upgrades frequentes
- papel do MySQL em alta disponibilidade
