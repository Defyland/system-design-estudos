# Capstones

Drills cumulativos para provar transferencia entre chapters.

## Como usar

1. responda em voz alta
2. primeiro em `modo entrevista`
3. depois em `modo trabalho real`
4. puxe os chapters so quando travar

## O que um capstone treina

- misturar areas sem virar salada
- escolher o que fica de fora
- responder com camadas: baseline, risco, evolucao
- distinguir o que Rails ja resolve do que Elixir ou Go ensinam melhor
