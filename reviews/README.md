# Reviews

Revisao espacada para consolidar os chapters sem depender de exercicio escrito.

## Como usar

1. responda tudo em voz alta
2. olhe o gabarito logo em seguida
3. classifique mentalmente cada card como:
   - `liso`
   - `hesitante`
   - `errei`
4. no proximo ciclo, gaste mais tempo no que ficou `hesitante` ou `errei`
